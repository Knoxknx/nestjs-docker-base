## Descripción

Estructura base de un proyecto usando NestJs con Typescript, Docker y Postgres.

## Instalación

```bash
$ npm install
```

## Ejecutando la app
### Local 
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
### Docker
```bash
# dev
bash run.sh dev

# prod
bash run.sh prod
```

## Versiones
* Node: 20
* Postgres: 15.1
